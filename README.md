# OnlineSCADA – Pre-Launch Landing Page

A modern, responsive pre-launch landing page for **OnlineSCADA**, built to showcase the platform and capture early user interest through an email waitlist.

## Features

- 🚀 Modern and responsive UI
- 📧 Email waitlist for early access
- 🔔 Notification signup for product launch
- ⚡ Fast and lightweight performance
- 🌐 Optimized for desktop and mobile devices

## Tech Stack

- React.js
